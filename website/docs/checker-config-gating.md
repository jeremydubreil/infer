Determines which config flags gate the execution of each line of code.

Activate with `--config-gating`.

Supported languages:
- C/C++/ObjC: No
- C#/.Net: No
- Erlang: No
- Hack: No
- Java: Experimental
- Python: No
- Rust: No
- Swift: No



## List of Issue Types

The following issue types are reported by this checker:
- [CONFIG_GATED](/docs/next/all-issue-types#config_gated)
- [CONFIG_UNGATED](/docs/next/all-issue-types#config_ungated)
